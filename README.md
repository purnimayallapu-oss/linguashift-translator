# 🌍 LinguaShift — Language Translation Tool

LinguaShift is a simple and stylish **Language Translation Tool** built using **HTML, CSS, and JavaScript**.  
It allows users to enter text, choose source and target languages, and get translated output instantly.

This project was created as part of my **CodeAlpha Internship Task**.

---

## 🚀 Features

- 🌐 Translate text between multiple languages
- 🔍 Auto-detect source language
- 🔁 Swap source and target languages
- 📋 Copy translated text
- ✨ Modern and responsive UI
- ⚡ Fast browser-based translation experience

---

## 🛠️ Tech Stack

- **HTML**
- **CSS**
- **JavaScript**
- **MyMemory Translation API**
